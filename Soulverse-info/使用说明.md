# 如何打开数字孪生生成器

## 方法一：使用Python服务器（推荐）

### macOS/Linux用户：

1. 打开终端（Terminal）
2. 进入项目目录：
   ```bash
   cd /Users/zousirui/Desktop/Soulverse分析/digital_twin_generator
   ```
3. 运行启动脚本：
   ```bash
   ./start_server.sh
   ```
   或者直接运行Python脚本：
   ```bash
   python3 start_server.py
   ```
4. 浏览器会自动打开，如果没有自动打开，请手动访问：`http://localhost:8000/index.html`

### Windows用户：

1. 双击运行 `start_server.bat` 文件
2. 浏览器会自动打开，如果没有自动打开，请手动访问：`http://localhost:8000/index.html`

### 停止服务器：

在终端/命令行窗口中按 `Ctrl + C`

---

## 方法二：使用其他HTTP服务器

如果你安装了Node.js，可以使用：

```bash
cd digital_twin_generator
npx http-server -p 8000
```

然后访问：`http://localhost:8000/index.html`

## 方法三：直接打开（可能有问题）

**注意：** 直接双击打开 `index.html` 文件可能会因为浏览器的安全策略导致API调用失败。

如果必须直接打开：
1. 双击 `index.html` 文件
2. 如果API调用失败，请使用方法一或方法二

## 常见问题

### Q: 提示端口被占用怎么办？
A: 修改 `start_server.py` 文件中的 `PORT = 8000` 改为其他端口（如8001、8080等）

### Q: 提示找不到Python？
A: 
- macOS/Linux: 安装Python 3: `brew install python3` 或 `sudo apt-get install python3`
- Windows: 从 https://www.python.org/downloads/ 下载安装

### Q: 页面打开了但是API调用失败？
A: 
1. 检查网络连接
2. 检查浏览器控制台（F12）是否有错误信息
3. 确认Gemini API Key是否有效

### Q: 页面样式显示不正常？
A: 确保 `styles.css` 文件与 `index.html` 在同一目录下

1. 打开终端/命令行
2. 进入 `digital_twin_generator` 目录
3. 运行：`python3 start_server.py`（macOS/Linux）或 `python start_server.py`（Windows）
